## Tools
- **Feast** as the feature store
- **Weights & Biases** for model experimentation, hyperparameter tuning, and metadata
- **Poetry** to build packages
- **Airflow** for data pipeline orchestration
- **Great Expectations** for data validation
- **Streamlit & Streamlit Community Cloud** for UI and frontend hosting
- **Railway** for backend hosting
- **FastAPI** for model serving using a REST API
- **Docker** for containerization
- **GitHub Actions** for Continuous Integration / Continuous Deployment (CI/CD)
- **pytest** for testing
